컨테이너 안에서 위젯들의 위치와 크기를 결정하는 객체

Grid : 격자 배치 관리자
표로 해서 나눌 수 있다. 
(ex)
0,0     0,1
1,0     1,1

Pack : 압축 배치 관리자
위젯들을 부모 위젯 안에 압축 가장 많이 사용

Plack : 절대 배치 관리자
